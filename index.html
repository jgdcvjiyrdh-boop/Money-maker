What is Termux?

Termux is an Android terminal emulator and Linux environment application that works directly with no rooting or setup required. It provides a powerful command-line interface on your Android device, allowing you to run thousands of Linux packages and development tools.

Essential Termux Tools

  • Git - Version control system for tracking code changes
  • Nmap - Network exploration and security auditing
  • OpenSSH - Secure shell for remote access
  • Python - Versatile programming language with extensive libraries
  • SQLite - Lightweight database management
  • Vim/Nano - Text editors for code and configuration
  • Net-tools - Network diagnostics and configuration
  • Hydra - Password cracking tool (for ethical use)

Getting Started

Installation

Termux is available on Google Play Store and F-Droid. After installation, run these commands to set up:

$ pkg update && pkg upgrade
# Update package lists and upgrade installed packages

$ pkg install git python nmap
# Install essential tools

$ termux-setup-storage
# Grant storage access to Termux

Basic Commands

Termux uses the same commands as standard Linux distributions:

$ ls
# List directory contents

$ cd folder_name
# Change directory

$ pkg search tool_name
# Search for available packages

Important Notes

Pro Tips

  • Use termux-chroot for a more complete Linux environment
  • Install termux-api to interact with Android features from terminal
  • Use pkg install proot to run distributions like Ubuntu
  • Backup your Termux environment with tar -zcf /sdcard/termux-backup.tar.gz -C /data/data/com.termux/files ./

Ethical Use

Termux provides powerful tools that should only be used ethically and legally:

  • Only test systems you own or have permission to test
  • Respect privacy and data protection laws
  • Use knowledge to improve security, not exploit vulnerabilities
  • Many tools are for educational purposes only

Sample Python Script for Termux

Here's a simple network scanner you can run in Termux (requires nmap installed):

#!/usr/bin/env python3
# Simple network scanner for Termux
import os
import sys

def main():
    # Get target IP from user
    target = input("Enter target IP: ")
    print(f"Scanning {target}...")

    # Run nmap scan
    os.system(f"nmap -sV {target}")
    print("Scan completed!")

if __name__ == "__main__":
    main()

Save this as scanner.py and run with python scanner.py in Termux.
